# C++ End Semester Lab Programs

This repository contains all C++ programs required for the end semester practical examination. The programs cover fundamental concepts of programming, object-oriented principles, and exception handling.

---

## 📌 Experiments Included

1. Linear Search
2. String Manipulation Operations
3. Swap Two Numbers Using Pointers
4. Swap Two Numbers Using Reference Variables
5. Store and Display Records Using Array of Objects
6. Store and Display Records Using Pointers to Array of Objects
7. Area / Volume of Shapes using Constructor Overloading
8. Divide by Zero Exception Handling
9. Array Out of Bounds Exception Handling

---

## 🧠 Concepts Covered

* Arrays and Searching Techniques
* String Handling
* Pointers and References
* Classes and Objects (OOP)
* Constructor Overloading
* Exception Handling in C++
